# コントーションスタジオ・ノガラ 本サイト・リニューアルデモ

既存サイト（https://www.contortion-jp.com/）のブランドイメージを継承しつつ、Shopifyベースのモダンな構成へと昇華させたリニューアル案のデモサイトです。

## 公開URL
**[デモサイトを表示する](https://shunking-tech.github.io/nugara-main-demo/)**

## リポジトリ情報
- GitHub Repository: [shunking-tech/nugara-main-demo](https://github.com/shunking-tech/nugara-main-demo)

## デザインコンセプト
- **既存ロゴの活用**: 実際のブランドロゴをヘッダー・フッターに配置。
- **柔らかい高級感**: コントラストを抑えたオフホワイト（#FAFAFA）とダークグレー（#222222）の配色。
- **実用的なCTA**: 体験レッスンへの動線を強調。
- **動的表現**: GSAPによる滑らかなスクロールアニメーション。
